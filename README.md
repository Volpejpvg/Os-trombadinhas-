# Os-trombadinhas-

Esse é o trabalho de programação na linguagem C do curso de engenharia de computação, a dupla que foi responsável por esse trabalho é João Pedro Venuto e Henry Biancchi

A parte responsável pelo menu foi o Henry, onde o mesmo ficou estudando junto do Arthur(Nosso colega de classe) e foi usado bastante o If Else e o Suitcase  para a organização do código 

No primeiro jogo, que também foi criado pelo Henry, ele teve algumas dificuldades na sua criação. O jogo não tem um placar mostrando a quantidade de acertos, mas ele funciona de forma correta e no final do primeiro jogo o mesmo aparece para sair ou voltar ao menu

Segundo jogo foi criado tanto por Henry na base da praticidade e por João Pedro que tentou ajudar mas não conseguiu dar muito suporte. No jogo dois foi usado a referência ao jogo do aluno Arthur(Da nossa classe) e também com ajuda do chatgpt(infelizmente a conversa com o chat foi perdida), tivemos algumas dificuldades como na organização do código e torná-lo funcional, ele começa mostrando o nome da tumba que o demos e um detalhe interessante que todos os personagens do jogo tem nome de alunos da nossa classe. Ele funciona de forma simples, o jogador escolhe uma caixa e nela aparecerá se achou o botão ou não.

No terceiro jogo foi feito integralmente por João Pedro, durante o processo do terceiro jogo foi usado bastante o chatgpt pois o mesmo não conseguiu achar tantas coisas para pegar de referência para fazer o jogo(principalmente o próprio jogo ele não achou na internet). Infelizmente a conversa do chatgpt foi perdida, mas temos dois links que um explica o que o código faz em cada parte e também outro para arrumar uma parte do código, a principal dificuldade do código foi no uso do locale.h que não funcionava direito no código
